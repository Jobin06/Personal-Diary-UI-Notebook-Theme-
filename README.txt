Personal Diary UI - Notebook Theme
===================================

Project Overview:
This is a personal diary application designed to mimic a physical notebook using HTML and CSS only. The design creates an authentic journal experience with realistic paper textures, handwriting-style fonts, and notebook-like styling.

Features:
- Notebook-style layout with spiral binding effect
- Paper-like background textures and colors
- Cursive and handwriting-style fonts (Google Fonts)
- Date-based entry system with individual cards
- Static navigation for months and recent entries
- Mood tracking indicators
- Responsive design for mobile and desktop
- Print-friendly styling
- Realistic shadows and paper effects

File Structure:
- index.html - Main diary page with all content
- css/style.css - Complete styling for notebook theme
- README.txt - This file with project information

Design Elements:
- Color scheme: Warm paper tones (cream, sepia, brown)
- Fonts: Dancing Script, Caveat, and Kalam from Google Fonts
- Textures: CSS gradients and patterns for paper effect
- Layout: Two-page spread mimicking an open notebook
- Effects: Spiral binding, page corners, realistic shadows

Technical Implementation:
- Pure HTML5 and CSS3 (no JavaScript)
- CSS Grid and Flexbox for layout
- Google Fonts for typography
- CSS gradients for textures and effects
- Media queries for responsive design
- Hover effects and smooth transitions

Browser Compatibility:
- Modern browsers (Chrome, Firefox, Safari, Edge)
- Mobile responsive design
- Print stylesheet included

Usage:
1. Open index.html in any modern web browser
2. Navigate through entries using the left sidebar
3. Scroll through diary entries on the right page
4. Enjoy the authentic notebook experience

Customization:
- Edit entry content in index.html
- Modify colors and fonts in css/style.css
- Add new entries following the existing structure
- Customize mood indicators and navigation items
